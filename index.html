¿De qué va?

Aprenderás y participarás en un proyecto Open Source con las más modernas tecnologías.

Internet of Things

Aprenderás a utilizar una Raspberry Pi y a programar diversos sensores en un proyecto basado en la IoT, totalmente Open Source.

Desarrollo web

Crearás una interfáz web para controlar el hardware y sus elementos, todo con las más modernas tecnologías y metodologías ágiles de desarrollo.

En la nube

Aprenderás a incluir integración continua, análisis de código o code reviews en tus proyectos de forma muy sencilla.

Estructura

El proyecto se dividirá en 4 pequeñas fases.

  • Febrero

    Intro básica

    Antes de empezar a tope hay que iniciarse! Tendremos varias sesiones en las que aprender las cosas básicas de Python, Git y Github para ponernos al día con la tecnología.

  • Primeras semanas de Marzo

    Diseño y estructura

    Hablaremos de planificación y desarrollo ágil, decidiendo a donde queremos llevar el proyecto, definiendo los objetivos y concretando las necesidades que tendremos.

  • Marzo - Mayo

    Hackatons

    Sesiones de hacking de la mano de tutores de gran nivel de empresas y asociaciones del sector con ganas de apoyar el software libre.

  • Marzo - Mayo

    Charlas

    También nos ofreceran diversas charlas en las que aprenderemos muchas cosas transversales al proyecto, APIs REST, Agile, LaTeX para documentación, Django Framework, etc

Comunidad de desarrollo

Queremos crear una comunidad de desarrolladores, por eso, cada semana tendremos una actividad muy práctica y enfocada en una tecnología libre, alrededor de este proyecto común disponible en Github para que todos podamos colaborar.

Hackatones

Pretendemos también hilar semanas en las que directamente tener un espacio disponible para realizar hackatones y avanzar en el desarrollo de la parte del proyecto que nos interese.

Software Libre

En este proyecto no solo se podrá colaborar asistiendo ya que al ser un proyecto software libre, el código estará publicado, así como los videos y la documentación de todas las charlas, por lo que podrás seguirlo y trabajar en el también a distancia.

Se encuentra publicada en el siguiente repositorio público de GitHub

Python

Un lenguaje para dominarlos a todos.

Python es un lenguage de programación sencillo de aprender y que te permite trabajar rápidamente e integrar sistemas de forma mas eficiente.

Django framework

El framework web para perfeccionistas con deadlines.

Django es un framework web para Python de alto nivel que facilita un desarrollo rápido y claro para que puedas enfocarte en programar tu app sin reinventar la rueda. Es libre y open source.

Git

Moderno y avanzado sistema de control de versiones.

Ligero, sencillo y tremendamente versatil para gestionar desde pequeños hasta enormes proyectos.

Todo el mundo usa Git hoy en día, ¿te lo vas a perder?

Github

La plataforma más famosa para el desarrollo colaborativo.

Aquí podrás almacenar todos tus desarrollos, aprenderás a utilizarlo basándote en Git, así como el flow de trabajo habitual en proyectos modernos.

De simple a avanzado

Trabajaremos con ramas, aprenderemos lo que es hacer code review, integración continua y a documentar un proyecto.

Raspberry Pi

Un miniordenador Open Source.

La Raspberry tiene múltiples usos, nosotros le conectaremos sensores y todo tipo de cacharros para obtener información y hacer un completo proyecto basado en la IoT (Internet of Things)

Travis CI

Integración continua Open Source.

Travis es un sistema de integración continua que básicamente te permite no liarla.

Tu subes tu código a tu repositorio de Github y Travis se encarga de pasarle todos los tests con diversas configuraciones e incluso de mandarte el resultado por Slack, Hipchat o por email.
